# PDF-Converter

# Can convert PDF files to some type of files: .docx, .jpeg, .jpg, .gif, .html, .png, .txt, .tiff.

## **_Libraries_ used in this project:**

- Pdf2Dom
- iText
- POI
- Pdf2Dom
